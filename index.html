JavaScript

The Language for Programming WebPages

JavaScript Example :

function greet(name) {
  let message = "Ni Hao, " + name + "!";
  return message;
}

const user = "Fyn Shit";
console.log(greet(user));

// Output: Ni Hao, Fyn Shit!

SQL

A Language for Accessing DataBases

SQL Example :

SELECT name, age
FROM Customers

ORDER BY name ASC;

-- Get customers older than 18
SELECT *
FROM Customers
WHERE age > 18;
